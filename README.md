# lemonade.dev

Personal portfolio and blog. Built with Astro + Go on Nix.

Live: <https://lemonade.dev> (準備中)

## Tech Stack

- **Frontend**: Astro + React Islands + Tailwind Islands
- **Backend**: Go + PostgreSQL
- **Infra**: Nix flake (devShell) + NixOS module (deployment)
- **Container**: Podman
